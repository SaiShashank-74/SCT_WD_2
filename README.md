# ⏱️ STOPWATCH

A futuristic and responsive stopwatch web application designed with a premium glassmorphic interface, smooth animations, and accurate time tracking functionality.

## 🚀 Project Overview

STOPWATCH is an interactive web application that allows users to accurately measure time intervals, record lap times, and manage stopwatch operations through an intuitive user interface.

Developed as part of **SkillCraft Technology Internship – Task 02**.

## ✨ Features

* Start Stopwatch
* Pause / Resume Stopwatch
* Reset Stopwatch
* Record Multiple Lap Times
* Circular Progress Ring Animation
* Glassmorphism UI Design
* Responsive Layout
* Keyboard Shortcut Support
* Smooth Hover Effects
* Futuristic Dashboard Theme
* Accurate Millisecond Tracking

## ⌨️ Keyboard Shortcuts

| Key   | Action          |
| ----- | --------------- |
| Space | Start / Pause   |
| L     | Record Lap      |
| R     | Reset Stopwatch |
